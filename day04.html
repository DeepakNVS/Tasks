Image Source

The src attribute of the <img> element specifies the source of the image. This can be a relative or absolute URL. Here are some examples:

  • src="image.jpg": This is a relative URL, and it assumes that the image is located in the same directory as the HTML file.
  • src="/images/image.jpg": This is an absolute URL, and it specifies the full path to the image, starting from the root directory of the website.
